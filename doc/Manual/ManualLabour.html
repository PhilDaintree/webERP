Labour Recording

Overview

The work done on work orders and contracts can be captured by the entry of time-sheets. Each employee can have a time-sheet.

Setting Up Employees

Labour in webERP is actually a special type of item - this allows the routings to be defined in the bill of materials along with all the other physical components. This requires that each employee have an associated labour type item which is previously defined in stock/item master. Labour items are simply stock items that have a stock category that is a labour type. Child items of labour type stock categories are all labour items.
